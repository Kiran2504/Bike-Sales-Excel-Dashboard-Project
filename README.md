# Bike-Sales-Excel-Dashboard-Project
Interactive Excel dashboard project for analyzing bike sales data using data cleaning, pivot tables, and visualizations. This project provides insights into customer demographics, income, commute distance, and purchasing behavior through an easy-to-understand dashboard built in Microsoft Excel.

# Project Overview

This Excel project focuses on creating an interactive dashboard for bike sales analysis using data cleaning, pivot tables, and visualizations. The dashboard provides useful insights into customer demographics and bike purchasing patterns in a simple and user-friendly format.

# Dataset

# Dataset Used – Bike Buyers Dataset on Kaggle
The project is based on the Bike Buyers 1000 dataset.

# Project Workflow
1) Data Cleaning
   Removed duplicate records
   Standardized categorical values such as marital status and gender
2) Working Sheet Creation
   Created a separate working sheet for data manipulation while keeping raw data unchanged
3) Pivot Tables
   Used pivot tables for summarizing and analyzing the dataset
4) Data Visualizations
   Built charts to represent important insights such as:
     Average income
     Commute distance
   Age brackets
5) Dashboard Creation
   Combined all charts and visualizations into a single interactive dashboard sheet
6) Chart Formatting
   Customized chart titles, labels, and formatting for better readability and presentation

# Dashboard Insights
  Average Income by Gender & Bike Purchase
     -Analysis of customer income based on gender and bike purchase behavior
  Customer Commute Distance
     -Distribution of customers according to commute distance and bike purchase status
  Customer Age Brackets
     -Bike purchase analysis across different age groups such as adolescent, middle-aged, and old customers
